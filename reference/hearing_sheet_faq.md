## 3. 回答の要件

### 3.1 回答のトーン
```
□ 丁寧・フォーマル（「〜でございます」）
□ やや丁寧（「〜です/ます」）
□ カジュアル（「〜だよ」）
□ 業務的（簡潔に事実のみ）
```

### 3.2 回答の長さ
```
□ 短い方がよい（1-2文）
□ 普通（3-5文）
□ 必要に応じて長くてもよい
□ 分からない / お任せ
```

### 3.3 回答に含めるべき情報
```
□ 結論（必須）
□ 条件（「いつ当てはまるか」）
□ 例外（「当てはまらないケース」）
□ 次の行動（「何をすればよいか」）
□ 参照元（「どの文書に書いてあるか」）
□ 連絡先（問い合わせ先の電話番号・メールアドレス）
□ URL・リンク
```

### 3.4 断定してよい範囲
```
■ 断定してよいもの:
  □ 手続きの方法
  □ 期限・締め切り
  □ 基準値・上限額
  □ その他: __________________

■ 断定してはいけないもの:
  □ 個別の判断結果
  □ 法的なアドバイス
  □ 人事評価に関すること
  □ 医療に関するアドバイス
  □ その他: __________________
```
