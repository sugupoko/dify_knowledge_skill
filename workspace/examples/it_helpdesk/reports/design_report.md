## 1. ナレッジ構造

### 全体構成

```
[ナレッジベース]
  ├── 層1: FAQ（70件） ← generated_faq.md から投入
  ├── 層2: 手順書（マニュアル本文をチャンク化）
  └── 層3: エスカレーションルール
```

### 層1: FAQ（直接回答層）

| 項目 | 内容 |
|------|------|
| 件数 | 70件 |
| ソース | it_manual.md（全10章）+ helpdesk_inquiries.md（トピック補完） |
| 生成パターン | パターンB（2ステップ — 質問先行→回答生成） |
| フォーマット | タブ区切り9列（No/カテゴリ/質問/回答/条件例外/次の行動/エスカレーション/参照元/言い換え） |
| ファイル | `data/generated_faq.md` |

#### カテゴリ別件数

| カテゴリ | 件数 | 優先度 |
|---------|------|--------|
| パスワード・アカウント | 9件 | 高（繰り返し問い合わせの典型） |
| PC・ハードウェア | 9件 | 高（件数最多） |
| セキュリティ | 9件 | 高（インシデント系含む） |
| VPN・ネットワーク | 8件 | 高（在宅勤務需要） |
| メール・Teams | 8件 | 高 |
| ソフトウェア・ライセンス | 7件 | 中 |
| クラウド・SaaS | 6件 | 中 |
| 障害・インシデント | 6件 | 高（エスカレーション必須） |
| プリンター・複合機 | 4件 | 中 |
| 申請・手続き | 4件 | 高（新入社員・異動時） |

### 層2: 支援文書

| 種類 | 内容 | チャンキング |
|------|------|------------|
| 手順書 | PCセットアップ（7ステップ）、パスワードリセット（5ステップ）等 | ステップ単位で分割 |
| 規程・ポリシー | パスワードポリシー、データ持ち出しルール、BYODルール | 条文・ルール単位 |
| 用語集 | GlobalProtect、スプリットトンネル、MDM 等の技術用語 | 1用語1チャンク |

### 層3: 運用情報

| 種類 | 内容 |
|------|------|
| エスカレーション | ランサムウェア→即座にLANケーブル抜く+内線9999。情報漏洩→上長+情報セキュリティ部門。 |
| 禁止事項 | 個人のパスワードを直接聞き出さない。法的判断をしない。人事・給与情報を回答しない。 |
| 連絡先 | ヘルプデスク: 内線9999 / helpdesk@example.co.jp |

## 2. チャンキング戦略

### Dify ナレッジベース設定

| 設定項目 | 推奨値 | 理由 |
|---------|--------|------|
| インデックス方式 | high_quality | Embedding ベクトル検索 |
| チャンクモード | automatic | FAQ形式なら1行=1チャンク |
| 検索モード | Hybrid (ベクトル + 全文) | キーワード不一致時もベクトルで補完 |
| Top K | 5 | FAQ 70件規模なら5で十分 |
| Score Threshold | 0.5 | 閾値が高すぎると取りこぼしが増える |
| Reranking | 推奨（利用可能なら） | 検索精度の向上に最も効く |

### Ollama 固有の注意

| 項目 | 推奨 |
|------|------|
| チャンクサイズ | 小さめ（128〜256トークン） |
| 結論先出し | 厳守（Lost-in-the-Middle対策） |
| ハルシネーション抑制 | プロンプトで「ナレッジベースにない情報は答えない」を明示 |

## 3. プロンプト設計

### システムプロンプト（推奨）

```
あなたは社内ITヘルプデスクのアシスタントです。
ナレッジベースに登録された情報に基づいて、従業員のIT関連の質問に回答してください。

## 回答ルール
- ナレッジベースの情報のみに基づいて回答してください
- 回答は最初の1-2文で結論を述べてください
- 条件や例外がある場合は明示してください
- 手順がある場合はステップ番号を付けてください
- ナレッジベースに該当情報がない場合は「この質問については、ヘルプデスク（内線9999）にお問い合わせください」と案内してください

## 緊急時の対応
以下のキーワードが含まれる場合は、通常の回答ではなく即座にエスカレーション案内をしてください:
- ランサムウェア / ウイルス / 不正アクセス → 「今すぐLANケーブルを抜いて、ヘルプデスク（内線9999）に電話してください」
- 情報漏洩 / 誤送信 → 「すぐに上長に報告し、情報セキュリティ部門に連絡してください」

## 禁止事項
- 個人のパスワードを聞き出す回答はしないでください
- 法的な判断・助言はしないでください
- 人事・給与に関する質問には回答せず、人事部への問い合わせを案内してください
```

### 開始メッセージ

```
ITヘルプデスクアシスタントです。
VPN、パスワード、PC、メール、ソフトウェアなどに関する質問にお答えします。
お気軽にご質問ください。
```

### サジェスト質問

```
- VPNに接続できないのですが
- パスワードを忘れました
- 新しいソフトウェアを申請したい
```
